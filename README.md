# Keys

- 'w / up_arrow' move up 
- 's / down_arrow' move down 
- 'a / left_arrow' move left 
- 'd / right_arrow' move right 
- 'f' lock camera to player 
- 'p' pause game 
- 'o' unpause game
- 'z' double speed 
- 'x' normal speed 
- '-' zoom out camera 
- '+' zoom in camera
- 'esc' close game
